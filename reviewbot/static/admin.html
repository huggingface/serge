Provider configs

Each config binds a provider + API key to a repository (or whole org with owner/*) and the GitHub users/orgs allowed to spend it. When a user submits a review, the most-specific matching config is picked; without a match the submission fails.

API keys are write-only — once saved, the value can't be displayed again. Edit a row to replace the key.
